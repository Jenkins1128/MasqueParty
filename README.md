# MasqueParty

Isaiah Jenkins

About:

MasqueParty is a social utility app that allows you to see people's bios and personalities at your current location.

## Technologies

* Swift
* CoreLocation
* Cocoapods
* Firebase Firestore
* Firebase Storage
* Firebase Auth
* Swift Package Manager
* Facebook SDK
* IQKeyboardManagerSwift
* Unit and UI tests

## Design Patterns

* MVC
* Delegation
